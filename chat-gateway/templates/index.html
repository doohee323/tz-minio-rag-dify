Chat Gateway

Unified chat API in front of Dify. Part of the tz-minio-rag-dify stack.

What this is

This service is the Chat Gateway: a single API and chat UI for multiple apps (e.g. DrillQuiz, CoinTutor). It sits in front of Dify, manages users and conversations, and can sync chat history to its own database for support and analytics.

Project: tz-minio-rag-dify

The full stack runs on Kubernetes: MinIO (object store), Qdrant (vector DB), RAG backends (CoinTutor / DrillQuiz), Dify (chatbots), and this Chat Gateway. RAG ingestion runs on a schedule; Dify apps call the RAG backends as tools.

GitHub source API docs (OpenAPI) Chat page Cache / history (API key required)

Chat page requires a valid JWT (?token=...). Use the chat from DrillQuiz or CoinTutor, or obtain a token via GET /v1/chat-token with X-API-Key.
